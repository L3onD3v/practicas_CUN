# 🚀 Bootcamp Análisis de Datos - Nivel Básico (Talento Tech / CUN)

¡Bienvenidos, campistas! Este repositorio es su centro de recursos para el entrenamiento intensivo en Ciencia de Datos. Aquí encontrarán todo lo necesario para transformar datos en decisiones inteligentes.

## 👨‍🏫 Tu Mentor
**León Ceballos Poveda** Apasionado por la tecnología, la enseñanza y el análisis de datos. Mi meta es guiarlos para que pasen de la teoría a la práctica real con Python y R.

---

## 🗺️ Estructura del Bootcamp (108 Horas)
El curso se divide en tres grandes pilares según nuestro Syllabus oficial:

1. **Módulo 1: Conceptos Fundamentales** (36h) - Estadística, ciclo de vida de los datos y tipos de análisis.
2. **Módulo 2: Herramientas de Software** (33h) - Python, R, bibliotecas esenciales y entornos de desarrollo.
3. **Módulo 3: Manipulación y Visualización** (39h) - Limpieza de datos (Preprocesamiento) y análisis descriptivo visual.

---

## 📂 Contenido del Repositorio
En este espacio encontrarán las siguientes carpetas:

* **/Datasets:** Datos reales para practicar en clase.
* **/Talleres:** Ejercicios prácticos, incluyendo el *Taller de Limpieza y Visualización* y el reto del *Detective de Datos*.
* **/Guias:** Material de apoyo como la guía de "Primeros Auxilios en Pandas" para solucionar errores comunes.
* **/Proyecto_Integrador:** Estructura y rúbricas para el desarrollo de su proyecto final.

---

## 🛠️ Herramientas Sugeridas
Para sacar el máximo provecho, asegúrense de tener instalado o acceso a:
* **Python 3.x** y **Jupyter Notebooks** (o Google Colab).
* **Librerías:** Pandas, Matplotlib, Seaborn.
* **Entornos:** Anaconda o RStudio.

---

## 📈 Evaluación y Rúbrica
Recuerden que el éxito en este bootcamp se mide bajo la escala de la CUN:
* **Cumple (90-100):** Comprensión sólida y aplicación precisa.
* **A mejorar (31-89):** Comprensión parcial con algunas dificultades técnicas.
